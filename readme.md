# Proyecto FrontEnd 1 - Sistema Escolar

Este proyecto consiste en un sistema básico de inicio de sesión desarrollado en JavaScript.

## Descripción

El sistema solicita un usuario y contraseña para permitir el acceso.

El programa:

- Solicita usuario, contraseña y rol
- Muestra los datos en consola
- Verifica si son correctos
- Permite máximo 3 intentos
- Bloquea el usuario después de 3 intentos fallidos
- Si el usuario y contraseña son correctos permite el acceso y se visualiza el dashboard.

## Instrucciones de uso

1. Abrir el archivo index.html
2. Llenar los datos de inicio de sesión
3. Seleccionar el rol 
4. Presionar el botón "Iniciar Sesión"
